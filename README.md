# Batch Encoder

![MainWindow](doc/MainWindow.png)

Encode multiple movie files super easy and ultra-fast. (with dedicated GPU)

This app is wrapper GUI for the [ffmpeg](https://ffmpeg.org/).

## Usage

1. Set options.
2. Drop file(s).

## Prerequisites

You need to have ffmpeg added to your PATH.
